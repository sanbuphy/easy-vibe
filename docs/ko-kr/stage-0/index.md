## 대상자

- 프로그래밍 경험이 없는 제품 관리자, 운영 직원
- 아이디어를 빠르게 검증하고 싶은 기업가
- AI 프로그래밍에 관심이 있는 비기술직 종사자
- 프로토타이핑 기술을 향상시키고 싶은 디자이너
